Observed Forecast (Random Forest)
How to read this: depth-to-water is measured in metres below ground level, so a rising line means the water table is sinking deeper — not recovering. Risk level is set from the long-run linear trend: ≥0.30 m/year decline is High, 0.10–0.30 m/year is Moderate, below that is Low. The Random Forest model uses the previous four quarters of readings plus seasonality to project forward; holdout error (last 8 quarters withheld) is shown per district in the accompanying notebook. This is a student project built on real CGWB data — treat the 3-year forecast as an illustrative trend extrapolation, not an operational prediction.
